# INFO442 Project: Netflix Recommender System

## 1. Project Overview
This project builds a recommender system tailored for Netflix users to suggest movies based on their preferences.
* **User:** Netflix users
* **Item:** Recommended movies

## 2. Project Plan & Requirements

### Phase 1: Data Preprocessing
* Aggregate data files
* Clean and prepare the dataset for modeling

### Phase 2: Model Training
* Implement a **Collaborative Filtering** model to generate recommendations

### Phase 3: Evaluation
* Assess model performance:
  * **Precision@K**
  * **NDCG@K**

### Phase 4: Deployment & Presentation
* Demonstrate the model in a **real-use** scenario
* Prepare final **presentation slides** detailing methodology and results
